# 🇹🇷 Türkçe

## 📌 Proje Hakkında

Bu proje, Python'da fonksiyon isimlerinin değiştirilebilmesini ve farklı isimlerle kullanılabilmesini göstermek amacıyla oluşturulmuştur.

Proje içerisinde bulunan örnek kodlarda fonksiyon isimleri değiştirilebilir. Fonksiyon adı değiştirildiğinde, kodun geri kalanında da yeni isim kullanılmalıdır.

Bu proje ile:

- Python fonksiyon mantığını öğrenmek,
- Fonksiyon isimlendirmeyi anlamak,
- Kodları kişiselleştirmek,
- Farklı kullanım örneklerini görmek

amaçlanmaktadır.

---

## ⚙️ Nasıl Çalışır?

Örneğin:

```python
def yazdir(girdi):
    print(girdi)

yazdir("Merhaba Dünya!")
```

Fonksiyon adı değiştirilebilir:

```python
def mesaj(girdi):
    print(girdi)

mesaj("Merhaba Dünya!")
```

Fonksiyonun yaptığı işlem değişmez, sadece çağırılma şekli değişir.

# 🇬🇧 English

## 📌 About The Project

This project was created to demonstrate that Python function names can be changed and customized.

The function names in the example codes can be modified. When a function name is changed, the new name must also be used in the rest of the code.

The goals of this project are:

- Learning Python function logic,
- Understanding function naming,
- Customizing code,
- Exploring different usage examples.

---

## ⚙️ How Does It Work?

For example:

```python
def print_text(input):
    print(input)

print_text("Hello World!")
```

The function name can be changed:

```python
def message(input):
    print(input)

message("Hello World!")
```

The function's operation stays the same, only the way it is called changes.
